# LegalConnect - Frontend Application

The modern, responsive frontend for LegalConnect, built with Next.js 15 and styled for a premium user experience.

## 🚀 Tech Stack
- **Framework**: Next.js 15 (App Router)
- **Language**: TypeScript
- **State Management**: Redux Toolkit & Redux Persist
- **Styling**: Tailwind CSS 4
- **Animations**: Framer Motion, Lottie (dotlottie-react)
- **Visuals**: React Globe.gl
- **Forms**: React Hook Form
- **API Client**: Axios

---

## ✨ Primary Features
- **Responsive Dashboards**: Custom interfaces for Admin, lawyers, and Users.
- **Smooth Navigation**: High-performance routing with Next.js.
- **Real-time UI**: Integrated with backend Socket.io for updates and chat.
- **Rich Visuals**: Interactive 3D globe and premium animations.
- **Form Validation**: Type-safe forms with React Hook Form.

---

## 🛠️ Getting Started

### Prerequisites
- Node.js (v18+)

### Setup
1. `npm install`
2. Create `.env` with:
```env
NEXT_PUBLIC_API_URL=http://localhost:8080/api
NEXT_PUBLIC_GOOGLE_CLIENT_ID=your_google_id
```
3. `npm run dev`
